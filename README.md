# Bounce Ball Game
## **Objective:**
To recreate the classic Bounce Ball game with simple obstacles and levels.
 # **Tools & Technologies:**
*  Java
*  OOP Concepts
*  Swing (GUI)
*  Timer
*  Exception Handling
# **Features:**
*  Ball movement control
*  Obstacle collision detection
*  Multiple levels
*  Score system
*  Game over screen
# Expected Outcome:
*  Understanding of game physics
*  Improved GUI development skills
*  Better event-driven programming knowledge
